	% --- Abstract ---
	\chapter*{ABSTRACT}
	\addcontentsline{toc}{chapter}{ABSTRACT}
	\thispagestyle{plain}
	
	The increasing demand for automation in logistics and indoor service environments has accelerated research in autonomous mobile robot navigation systems. This project presents the design and implementation of a simulation-based autonomous indoor parcel delivery system developed using Robot Operating System 2 (ROS 2) and the Gazebo simulation platform. The entire system is modeled, implemented, and evaluated in a virtual environment without the use of physical hardware. The primary objective of the project is to develop a reliable navigation framework that enables a differential-drive mobile robot to autonomously plan and execute parcel delivery tasks within structured indoor environments such as university buildings, offices, and hospitals. An Improved A* (IA*) algorithm is employed for global path planning to generate collision-free routes on occupancy grid maps. To address the discontinuities and sharp turns produced by grid-based planners, spline-based path smoothing techniques are implemented to generate continuous and kinematically feasible trajectories. A Model Predictive Control (MPC) strategy is designed to ensure accurate trajectory tracking while respecting motion constraints and maintaining system stability. The system is fully integrated within the ROS 2 architecture, utilizing standard communication mechanisms for localization, mapping, navigation, and control. A web-based interface is developed to allow users to assign delivery goals and monitor robot behavior in real time during simulation. Performance evaluation in multiple simulated indoor scenarios demonstrates reliable navigation, effective obstacle avoidance, and smooth trajectory execution. This work provides a scalable software foundation for autonomous indoor delivery systems and establishes a framework for future hardware deployment and real-world implementation.
