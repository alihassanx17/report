\section{Overview}
This paper proposes a simulation-based autonomous indoor parcel delivery system. Built on a mature development tool stack, the system has an intelligent mobile robot capable of autonomous navigation as its core configuration, and adopts a full-simulation development model, which lays a clear foundation for the subsequent elaboration of technical details.

The core tasks of the differential drive robot navigation system in this study include mapping, localization, path planning, trajectory smoothing, and obstacle avoidance. For global path planning, this study adopts an improved A* algorithm, which generates collision-free paths based on an occupancy grid map. To address the problem that grid-based paths have sharp turns that differential drive robots cannot follow smoothly, a spline smoothing technique is introduced. A model predictive trajectory tracking module will be developed in subsequent work.

The whole system works inside the ROS 2 framework. It uses ways of talking like topics and services to work together on navigation and control. The system also has a web-based control interface. This interface allows the user to assign tasks to the robot, such as delivering items, and set the robot's destination. Users can also view what the robot is doing in real time while testing it.

The project uses only simulation so it is safe to test the robots navigation without using a real robot. This is also useful if we want to use a robot in the real world. The simulation and the ROS 2 framework allow us to verify the robot’s behavior before deploying it in the real world.
